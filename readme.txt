# Videograph | Template

This project is a HTML template named "Videograph." The template is designed for videographers and video production services, featuring various sections such as Header, Hero, Services, Work Gallery, Counter, Team, Blog, Call to Action, and Footer. The template also includes interactive elements such as a video popup, services, and team sections, and a blog carousel.

## Setup

1. Clone the repository or download the source code.
   ```bash
   git clone https://github.com/yourusername/videograph-template.git
   ```

2. Open the project directory.
   ```bash
   cd videograph-template
   ```

3. Open `index.html` in your preferred web browser to view the template.

## Usage

### Header

The header contains a logo and a navigation menu. The navigation menu includes links to different sections of the page and a dropdown menu for additional pages.

### Hero Section

The hero section features a carousel with background images and text overlays. It includes a call-to-action button.

### Services Section

The services section showcases the various services offered. It includes descriptions and icons for each service.

### Work Gallery

The work gallery displays a grid of images with video popups. Clicking on an image opens a video in a popup.

### Counter Section

The counter section displays statistics with icons and numbers. 

### Team Section

The team section introduces the team members with their photos, names, and roles. It also includes social media links for each team member.

### Blog Carousel

The blog section features a carousel of blog posts with titles, dates, comments, and short descriptions.

### Call to Action

The call-to-action section encourages users to start their stories with a heading, description, and a button.

### Footer

The footer contains a logo, social media links, about us section, navigation links, and a newsletter subscription form.

## Customization

To customize the template, you can modify the HTML, CSS, and JavaScript files as needed. Here are some common customizations:

- **Change Logo**: Replace the `src` attribute of the `<img>` tag inside the `.header__logo` div in the header section.
- **Update Contact Info**: Modify the contact information in the footer section.
- **Change Services**: Update the content inside the services section to reflect the services you offer.
- **Add/Remove Work Items**: Add or remove `<div>` elements inside the work gallery section as needed.
- **Modify Colors and Fonts**: Update the CSS files to change the colors and fonts.
